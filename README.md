# Belle (Elm)

# Development

## Setup

```
elm package install
```

## Formatting

We are using elm-format, read here how to set it up: https://github.com/avh4/elm-format

```
elm-format src/
elm-format examples/
```

# Run the Examples

```
cd examples
elm package install
elm reactor -a=localhost
```

Visit: `http://localhost:8000/Accordion.elm` in your browser
